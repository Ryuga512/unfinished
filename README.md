専門学校時代に作ったゲームです。

ファイルが2つ含まれています

1つは「unfinished」という2Dゲーム（名前は「unfinished_refactoring」になっています）

プレイヤー移動：矢印キー
攻撃：スペースキー
ダミーの移動：WASD
ダミーへ瞬間移動：Z

2つ目は「影撃ち」という3Dシューティングゲーム（名前は「shadow_refactoring」になっています）

プレイヤー移動：WASD
攻撃：スペース
視点移動：マウス

影を撃つと緊縛状態になり、その状態でぶつかると多くのポイントを獲得できます
ただぶつかってもポイントは発生せず、ただ撃っても得点は低いです
